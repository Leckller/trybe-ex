Div | Span

- Temos as tags de conteiner e linha, respectivamente se chamam "div" e "span", onde a tag "div" cria um conteiner separando os itens um em cima do outro, já o "span" separa dentro de uma linha atribuindo um valor apenas onde está a tag e seguindo normalmente a string após o fechamento da tag "span".
Segue o exemplo:

Meu nome é Gusttavo
Ruy
Nascimento Ferreira

Meu nome é Gusttavo Ruy Nascimento Ferreira









- A tag "style" (utilizada no head) serve como um "documento de css" para fazer estilizar o html no próprio doc html (para melhor organização aconselho a separar as partes de javascript e css em outros arquivos)

- A tag "id" funciona de maneira única, em outras palavras o id só pode corresponder a uma tag, já a tag "class" funciona para mais de uma tag.
Segue o exemplo, meu rei:

id class class 2







- Temos as seguintes tags para estilo em css, "background-color" para mudar a cor do fundo da string, da mesma forma a tag "color" também muda a cor mas dessa vez apenas a cor do texto.

- Temos a tag de espaçamento entre as linhas, chamada de "line-height"

- Temos outras tags de texto "font" e suas variantes, em especial as seguintes versões "font-size" que serve para escolher o tipo de fonte do texto, a "font-weight" que serve para configurar o peso do texto (em outras palavras a grossura das letras), a "font-size que define o tamanho da fonte, e a "font-variant" que serve para modificar a forma em que as letras irão aparecer.

- Temos duas tags de espaçamento, a tag "padding" que faz espaçamentos no interior da string e a tag "margin" que é utilizada para dar espaçamento ao redor da string. (Lembrando que há várias formas de definir o tamanho, o mais comum é o px {pixels})

- Temos as tags "text" julgo que as seguintes são as mais importantes, a tag "text-align" é utilizada para configurar a posição da string, também temos a tag "text-decoration" que serve para configurar a decoração do texto ou para tirar a decoração de um link que por padrão vem com "underline".

- Temos as tags h1,h2,h3,h4,h5,h6. Que de forma crescente diminuem o tamanho da fonte.

- Temos a tag "p" que faz parágrafos.

- Temos a tag "br" que corta a linha a partir dela.

- Para mudança de estilo do texto temos os comandos "strong" para deixar em negrito e "em" para deixar o texto em italico.

- Temos as tags de listas ordenadas e desordenadas sendo elas,respectivamente: "ol" e "ul", para abrir uma linha na lista utilizamos a tag "li"

Com imagens fica mais nítido a diferença do overflow-x. é uma questão de lógica mesmo, as imagens não se alteram com a altura e sim nos dois lados para não perder resolução, logo é algo mais fixo e não se altera com mais linhas, já os textos se dividem mais com a altura de acordo em que a largura(width) é alterada.

Temos varias funções da tag "border", como a "border-width" para controlar a largura da borda, tendo o valor "medium" por padrão. Temos também a "border-style" por padrão vem configurada como "none", por isso não vemos nd enquanto não alterar essa propriedade. Também temos o "border-color" que tem como padrão o "currentcolor", esse valor deixa a cor da borda como o mesmo do elemento.

Temos a tag "display" que serve para alterar a maneira como os conteiners se comportam dentro de uma página, como alterar o comportamento de uma div que é por "bloco" para o comportamento de um span que é "inline".

Temos as tags de espaçamento dos elementos, como a tag "position" que pode ter os valores "fixed" para fixar aquele objeto na tela de forma que possa "scrollar" e aquele objeto seguir fixo naquela parte. Também temos o valor "absolute", que de forma simples deixa aquele objeto como absoluto e posicionado em primeiro e à cima dos demais elementos (Lembrando que as img são por padrão "inline"). Para solucionar esse problema dos objetos um em cima do outro devido o valor absolute temos a tag "z-index" que com valores numerados colocamos quem aparece primeiro(embaixo de todos) a partir do 1. Também temos a tag "float" para flutuar elementos na página, fazendo com que os demais elementos, como os textos, o contornem.

Seguinte, temos algumas maneiras de combinar classes, e temos uma maneira de especificar a parte em que queremos estilizar através de descendentes. Podemos utilizar mais de uma classe nas tags de html, apenas separando elas com um "espaço" dentro da class, por exemplo:
class="modelo-1 modelo-2"
dessa forma colocamos duas classes com configurações diferentes em uma tag. Agora dentro do CSS, podemos especificar o local no qual iremos estilizar, utilizando uma lógica de "descendentes", por exemplo:
Uma lista ordenada que tem linhas e um paragrafo, para especificar ela fazemos numa ordem de pai filho neto -> -> -> ol li p
Dessa forma especificamos que para todos os paragrafos que estão dentro de uma "li" que está dentro de uma "ol" tenham os valores que serão adicionados dentro das chaves.
Extra! podemos utilizar mais especificações utilizando classes, da seguinte forma:
ol.modelo-1 li p
Dessa forma especificamos que para todas os paragrafos que estiverem dentro de uma linha e que estiverem dentro de uma lista ordenada e que tiverem a class "modelo-1" recebam os valores dentro das chaves. Mo role isso lendo, mas é bem simples na prática.
