- Temos as tags de conteiner e linha, respectivamente se chamam "div" e "span", onde a tag "div" cria um conteiner separando os itens um em cima do outro, já o "span" separa dentro de uma linha atribuindo um valor apenas onde está a tag e seguindo normalmente a string após o fechamento da tag "span". Segue o exemplo:

Meu nome é Gusttavo
Ruy
Nascimento Ferreira

Meu nome é Gusttavo Ruy Nascimento Ferreira



























- A tag "style" (utilizada no head) serve como um "documento de css" para fazer estilizar o html no próprio doc html (para melhor organização aconselho a separar as partes de javascript e css em outros arquivos)

























- A tag "id" funciona de maneira única, em outras palavras o id só pode corresponder a uma tag, já a tag "class" funciona para mais de uma tag. Segue o exemplo, meu rei:

id class class 2

























- Temos as seguintes tags para estilo em css, "background-color" para mudar a cor do fundo da string, da mesma forma a tag "color" também muda a cor mas dessa vez apenas a cor do texto.

























- Temos a tag de espaçamento entre as linhas, chamada de "line-height"

























- Temos outras tags de texto "font" e suas variantes, em especial as seguintes versões "font-size" que serve para escolher o tipo de fonte do texto, a "font-weight" que serve para configurar o peso do texto (em outras palavras a grossura das letras), a "font-size que define o tamanho da fonte, e a "font-variant" que serve para modificar a forma em que as letras irão aparecer.

























- Temos duas tags de espaçamento, a tag "padding" que faz espaçamentos no interior da string e a tag "margin" que é utilizada para dar espaçamento ao redor da string. (Lembrando que há várias formas de definir o tamanho, o mais comum é o px {pixels})

























- Temos as tags "text" julgo que as seguintes são as mais importantes, a tag "text-align" é utilizada para configurar a posição da string, também temos a tag "text-decoration" que serve para configurar a decoração do texto ou para tirar a decoração de um link que por padrão vem com "underline".

























- Temos as tags h1,h2,h3,h4,h5,h6. Que de forma crescente diminuem o tamanho da fonte.


























- Temos a tag "p" que faz parágrafos.

























- Temos a tag "br" que corta a linha a partir dela.

























- Para mudança de estilo do texto temos os comandos "strong" para deixar em negrito e "em" para deixar o texto em italico.

























- Temos as tags de listas ordenadas e desordenadas sendo elas,respectivamente: "ol" e "ul", para abrir uma linha na lista utilizamos a tag "li"
